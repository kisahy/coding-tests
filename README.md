# 코딩 테스트

이 프로젝트는 **프로그래머스 코딩 테스트 문제**를 Kotlin으로 직접 풀어보고 개인적으로 정리한 저장소입니다.

문제별 풀이와 테스트를 Gradle 환경에서 확인할 수 있습니다.
